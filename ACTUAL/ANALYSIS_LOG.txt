WHAT THIS IS
  The derivation history of the theoretical model -- every consequential change
  to the utility / impaired-miles / crash-probability structure, WHY it changed,
  and what it bought. PART A logs empirical decisions; this logs MODEL decisions
  (the crosswalk previously punted these to the README -- this is their home now).
  Placed first because the theory is written first in the manuscript.
  DATES: the source transcripts are undated, so entries are SEQUENCE-accurate but
  calendar-APPROXIMATE -- fix to absolute dates if chat timestamps are recovered.
  Append newest at the bottom; mark superseded model forms, do not delete them.

  END-STATE MODEL (what the paper carries): quasi-linear utility with an
  accident-disutility term; purely multiplicative impaired miles m = gamma*a*d
  with d = delta_c * n; crash object lifted from individual pi_i to county Pi_c
  with an inbound (attractor) term. Entries T1-T8 show how each piece arrived.

[2026-03] T1  Initial spatial travel-cost setup (per-unit cost, additive miles)
  MODEL:     U_i = u(a_i) + x_i ;  budget  p*a_i + tau(d_i)*a_i + x_i = y_i
             (tau a PER-UNIT travel cost -> effective price p* = p + tau(d_i)).
             D_c in {0,1}; d_i = 0 if wet, dbar_i > 0 if dry.
             Crash prob pi_i = pi(a_i, m_i); impaired miles m_i = alpha*a_i + beta*d_i.
  WHY:       Hotelling travel-cost + quasi-linear utility (kills income effects,
             so policy responses are pure price effects). alpha*a_i = impaired
             miles scaling with consumption; beta*d_i = the return trip from a
             distant purchase.
  PREDICTS:  a^dry < a^wet (Prediction 1); net crash effect ambiguous
             (consumption + vs distance -).
  STATUS:    superseded -- per-unit cost by T2, additive miles by T3 (see R in PART R).

[2026-04] T2  Fixed trip cost replaces per-unit travel cost
  CHANGE:    budget -> p*a_i + T(d_i) + x_i = y_i ;  effective per-unit price
             p* = p + T(d_i)/a_i, which FALLS with quantity.
  WHY:       A purchase trip is undertaken once regardless of quantity, so trip
             cost is fixed, not per-unit; a per-unit cost that rises with quantity
             is economically wrong. Side effect: builds in stockpiling (dry-county
             buyers make fewer, larger trips) -- carried forward as a reason
             aggregate VMT can be quiet even when the mechanism is real (see T7).
  STATUS:    current.

[2026-04] T3  Additive -> multiplicative impaired miles
  CHANGE:    m_i = alpha*a_i + gamma*a_i*d_i  (distance enters ONLY interacted
             with consumption).
  WHY:       The additive beta*d_i term is harm-relevant only if the driver is
             impaired on the return leg -- i.e. it is alpha*a_i in disguise unless
             distance is conditioned on having consumed. Off-premise retail is the
             treatment margin (drink at/after purchase), so the dangerous return
             trip is jointly determined by a and d. Multiplicative form makes
             distance AMPLIFY the risk of a given consumption level rather than
             generate risk independently.
  KEPT BOTH (transitional): alpha*a_i (impaired miles regardless of purchase
             location; persists after wet) vs gamma*a_i*d_i (return-trip miles;
             eliminated as d -> 0).
  STATUS:    superseded by T4 (alpha absorbed).

[2026-04] T4  Distance normalization d = delta_c * n ; collapse to pure multiplicative
  CHANGE:    d_i = delta_c * n_i, with delta_c = 1 (wet) or > 1 (dry, scaling with
             county boundaries crossed) and n_i = within-county distance. Then
             m_i = a_i(alpha + gamma*delta_c*n_i), which at delta_c=1 shows alpha is
             redundant -> COLLAPSE to  m_i = gamma*a_i*delta_c*n_i = gamma*a_i*d_i.
  WHY:       d does NOT drop to 0 when a county goes wet -- residents still drive
             within-county to the store (user objection). Normalizing within-county
             travel to n and letting dry status amplify via delta_c fixes this and
             unifies everything under one mechanism: impaired miles scale with
             consumption and distance; dry status amplifies distance.
  STATUS:    current (this is the headline impaired-miles spec).

[2026-04] T5  Accident-disutility term (Becker internalization)
  CHANGE:    U_i = u(a_i) + x_i - lambda*pi(a_i,m_i)*kappa ;  kappa = accident cost,
             lambda in [0,1] = internalization. 0<lambda<1 = partial internalization
             -> interior solution, links to Becker (1968).
  WHY:       Justifies why consumers don't drink to oblivion; supplies a marginal
             condition. Rejected the alternative expected-legal-cost form (-p(m)*F)
             as pulling toward structural estimation beyond scope -- kept only as a
             mentioned extension.
  PAPER:     Beamer slide 7 utility + variable table updated to this form.
  STATUS:    current.

[2026-04~05] T6  Comparative-static sign discipline (multiple corrections)
  ISSUE:     Differentiating w.r.t. dryness D_c while LABELLING channels by "going
             wet" produced repeated sign errors. Product-rule expansion of
             gamma*a*d gives THREE terms: consumption, distance-direct, and a
             distance-consumption interaction.
  RESOLUTION (final, correct): the full chain rule KEEPS dpi/dm in front of both
             distance terms (an earlier claim that it "absorbed out" was wrong):
               d(pi)/dD_c = dpi/da * da/dp* * dp*/dD_c                    (consumption)
                          + dpi/dm * gamma*n*a * ddelta_c/dD_c            (distance-direct)
                          + dpi/dm * gamma*n*delta_c * da/dp* * dp*/dD_c  (interaction)
             Signs: ddelta_c/dD_c > 0 (dry raises delta), da/dp* < 0, dp*/dD_c > 0.
             The travel-distance effect is itself AMBIGUOUS (its two components
             oppose), so the static has three competing forces, not two.
  WHY IT MATTERS: the ambiguity IS the paper's tension; the earlier "distance
             unambiguously negative" framing was an artifact of the sign slips.
  LESSON:    state the derivative-wrt-D_c and the channel labels in ONE consistent
             frame; keep dpi/dm so all three terms are in probability units.
  STATUS:    current.

[2026-06] T7  VMT as observable proxy for the distance channel (NOT a swap for m)
  DECISION:  Keep m_i LATENT. Do NOT (a) try to measure impaired miles (no data;
             inferring them from FARS crashes is circular), or (b) substitute VMT
             for m_i (severs the link to pi and, worse, VMT is blind to the
             consumption term alpha*a since people drink near home).
  MAPPING:   V_c = V_base + V_acq; the acquisition component's impaired return leg
             is the empirical shadow of gamma*a*d. VMT tests the DISTANCE channel
             only; the crash ATTs carry the net effect. It DISCRIMINATES H1
             (low-baseline-traffic) from H2 (attractor: inbound VMT rise).
  CAVEAT:    DVMT is total travel, not impaired miles; signal-to-noise is rough and
             HPMS rural roads are heavily modeled -> expect attenuation.
  PAPER:     Mechanisms. See PART A #11 and VMT_NOTES.txt.
  STATUS:    current.

[2026-06] T8  Lift crash object from individual pi_i to county Pi_c (attractor becomes native)
  CHANGE:    Realized crashes defined at the COUNTY of crash (as FARS records and
             as C&S-A estimates):  Pi_c = Pi(A_c, M_c), with
               M_c = gamma*a_own*delta_c*n                            (own residents, falls as delta_c -> 1)
                   + sum_{j in nbrs} gamma*a_j*d_{j->c}*1(c wet)      (inbound, jumps from 0).
  WHY:       At the individual level an inbound driver's crash has nowhere to live;
             defining Pi at the county adds a FOURTH, positive term (the inbound
             jump) that the single-agent static structurally cannot contain. Large
             catchment (rural, dry-surrounded) -> inbound dominates -> net POSITIVE;
             small catchment (urban) -> own-resident reduction wins -> NEGATIVE. The
             rural/urban reversal becomes a PREDICTION, not an anomaly -- this is
             the paper's spatial contribution.
  DISCIPLINE: two-layer model -- keep individual pi_i in utility (behavior chosen
             against own perceived risk); county Pi_c is the realized, measured
             outcome. On aggregation (pi nonlinear -> Jensen), define Pi_c as a
             county hazard over county exposure directly (representative-county
             primitive), stated as a modeling choice, not derived from exact
             micro-aggregation.
  FRAMING:   Do NOT write "the model was wrong and we patched it." Write "the
             within-county prediction is correct and is SWAMPED by a cross-county
             externality the prior literature ignored." The extension is principled
             (not post-hoc) because it makes NEW checkable predictions: the
             inbound-VMT signature (T7) and the symmetric spillover pattern
             (PART A #10 + spillover).
  PAPER:     Theory Parts 1-2; motivates heterogeneity (#15) and the attractor tests.
  STATUS:    current.
